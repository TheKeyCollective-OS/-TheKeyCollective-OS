# The Key Collective OS — Executive Refinement Sprint 5

A deployment-ready static personal operating system for GitHub Pages.

Open `index.html` through a web server or deploy the folder contents to GitHub Pages. This build uses browser-local storage and IndexedDB; no private server or API key is required.

See `SPRINT_5_RELEASE_NOTES.md` and `SPRINT_5_REGRESSION_CHECKLIST.md`.
